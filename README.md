## Publicación automática en GitHub Pages

Este paquete incluye scripts de Windows para crear el repositorio, subir los archivos y activar GitHub Pages.

### Primera publicación

1. Descomprime todo el contenido en una carpeta normal.
2. Ejecuta `01_INSTALAR_REQUISITOS.bat`.
3. Cierra la ventana y abre nuevamente la carpeta.
4. Ejecuta `02_PUBLICAR_EN_GITHUB.bat`.
5. Inicia sesión en GitHub cuando se abra el navegador.

El script crea automáticamente un repositorio público llamado:

`portafolio-jesus-colbert`

La dirección esperada será:

`https://TU-USUARIO.github.io/portafolio-jesus-colbert/`

### Actualizaciones posteriores

Después de modificar `index.html` o reemplazar el CV, ejecuta:

`03_ACTUALIZAR_GITHUB.bat`

GitHub Actions volverá a publicar el sitio automáticamente.

### Otros archivos

- `00_ABRIR_PORTAFOLIO.bat`: abre el portafolio localmente.
- `04_VER_ESTADO_Y_ABRIR.bat`: muestra las últimas publicaciones y abre GitHub.
- `CONFIGURACION.bat`: permite cambiar el nombre del repositorio y los datos de Git.
